Features

Expression Search / GMailUI - NG is an extension to Thunderbird which adds powerful message searching features. Init by Ken Mixter, developed over many years by Opera Wang, updated to TB 78/91/102+ by Klaus Buecher/opto.

  • Type "from:fred to:tom attachment:yes" or "f:fred t:tom att:yes" to see all messages from Fred to Tom in the current view that have an attachment.
  • Type "hello world" to search within the current view for this string in subject/from/to/cc.
  • Type "s:bbb t:(oo -pp)" to see all messages with subject bbb, send to oo but not to pp.
  • Press Ctrl+Enter to have this search apply to all folders through virtual folder.
  • Press Shift+Enter to invoke Gloda Search in all indexed folders.
  • Use Ctrl/Shift+Right Click to search the clicked subject/From/Recipient.

This add-on is constantly updated for newer versions of Thunderbird and it is gratefully appreciated if you can support my work with a donation.

New Features

Fulltext search for XNotes: If XNotes++ (> version 3.5) is installed, which allows to add notes to emails, then:
xn:* all emails with XNotes, -xn:* all emails without XNotes, xn:Peter all emails, where the XNote has Peter in it, -xn:Peter XNote has no Peter in text,
^xn:Peter all emails where the XNote starts with Peter (in a company, the notes could start with the relevant employee's name),
^xn:Peter xn:contract f:Mozilla all emails from Mozilla, where Peter made a note on a contract
and many more search possibilities. XNote++ on Addons.thunderbird.net, v3.5 will follow soon.

Search for Unreplied Messages: New Quickfilter for unreplied messages as button in the quickfilterbar. Combine with Star or Contact or 'from: sender' (in the search textbox) to reduce number of hits, or use Bookmarks addon.

Dark Mode: Adapt tooltip colours to Dark Mode. Needs TB restart after changing theme.

Search query builder: Clicking the browserAction button displays a popup with buttons for nearly all searchoperations. Click the button -e.g. 'to', then enter part of the name or email address. Follow by 'age', then enter number of days. Correspondingly for the actions you need in your search.
The up/down buttons scroll through the message list, not needed to make mouse leave this window. But at least one message needs to be highlighted for that.
