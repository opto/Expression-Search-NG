Features

Expression Search / GMailUI - NG is an extension to Thunderbird which adds powerful message searching features. Init by Ken Mixter, developed over many years by Opera Wang, updated to TB 78+ by Klaus Buecher/opto.

  • Type "from:fred to:tom attachment:yes" or "f:fred t:tom att:yes" to see all messages from Fred to Tom in the current view that have an attachment.
  • Type "hello world" to search within the current view for this string in subject/from/to/cc.
  • Type "s:bbb t:(oo -pp)" to see all messages with subject bbb, send to oo but not to pp.
  • Press Ctrl+Enter to have this search apply to all folders through virtual folder. (coming back soon)
  • Press Shift+Enter to invoke Gloda Search in all indexed folders. (coming back soon)
  • Use Ctrl/Shift+Right Click to search the clicked subject/From/Recipient. (coming back soon)

This add-on is constantly updated for newer versions of Thunderbird and it is gratefully appreciated if you can support my work with a donation.
